---
title: "Contrato"
description: "Employment contract assigned to a teacher, specifying salary, hours, and duration."
---

`Contrato` represents the employment agreement between NOUS Academy and a teacher. It defines the teacher's base salary, minimum work hours, additional-hour rate, and contract duration. The academy's capital is reduced by the total salary cost when a contract is created. Each contract receives a sequential auto-generated code.

## EstadoContrato values

| Value | Meaning |
|---|---|
| `ACTIVO` | Contract is in force |
| `SUSPENDIDO` | Contract is temporarily suspended |
| `FINALIZADO` | Contract has ended; the teacher's `contrato` field is set to `null` |

## Validation rules

| Exception | Condition |
|---|---|
| `FechaInvalida` | `fechaInicio` is `null` or before today |
| `ValoresNegativos` | `salarioBase â‰¤ 0`, `horasMinimas < 0`, `costoHoraPrivadaAdicional < 0`, or `meses â‰¤ 0` |
| `EstadoContratoInvalido` | `nuevoEstado` is `null` in `actualizarEstadoContrato()` |
| `DineroInsuficienteParaContratar` | Academy capital is less than `salarioBase * meses` (checked by the controller before creating the contract) |

## Commercial impact

`Contrato` implements `ComportamientoContable`. The `impactoComercial()` method returns `-(salarioBase * meses)`, a negative value that reduces the academy's capital when the controller processes the contract.

```java
// A contract for 2 months at 3,000,000/month
new Contrato(LocalDate.of(2026, 4, 1), 3_000_000, 40, 50_000, 2);
// impactoComercial() = -(3_000_000 * 2) = -6_000_000
```

## Finalization behavior

When a contract is transitioned to `FINALIZADO` via `Docente.actualizarEstadoContrato()`, the method additionally sets the teacher's `contrato` reference to `null`. This ensures the teacher no longer has an active contract in memory even though the contract record is preserved in `historialContratos`.

## File persistence

Contracts are not persisted as standalone files. They are serialized as part of the `Docente` object (stored in the `.doce` file). The method `copiarFicheroContrato(String dir)` is available for standalone use.
